Privacy Policy

1. Introduction

Your privacy is critically important to us. This Privacy Policy explains how "thx/chat" handles your personal information.

2. Data Collection and Use

"thx/chat" does not collect, store, or process any personal data. We operate on a non-extractive model where no user data is collected or stored.

3. Message Deletion

By default, "thx/chat" deletes any message from the unit's memory after a short pre-defined period of time (1 minute). No messages are stored on the server side.

"thx/chat" uses randomly generated user profiles. However, it is not an anonymization tool and does not hide communication patterns over the network.

5. Open Source and User Autonomy

"thx/chat" is open-source and part of the fediverse concept, which aims to avoid monopolies on services and provide autonomy to small "nodes" in a larger network. The implementation of "fedi" protocols is part of the next development phase of "thx/chat".

This feature aims to help users understand digital communication and provide insights for other contexts.
